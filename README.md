# Modern Shopify Theme

A clean, modern Shopify theme inspired by contemporary e-commerce designs.

## Features

- Responsive design
- Modern CSS with gradients and shadows
- Section-based architecture
- Customizable hero banner
- Product grid section

## Installation

1. Download the theme files
2. Upload to your Shopify store
3. Customize sections in the theme editor

## Customization

Edit the sections in the Shopify theme customizer to change colors, text, and layout.
